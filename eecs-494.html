EECS 494 P1 - Zelda Recreation

A Unity recreation of the first dungeon from The Legend of Zelda (1986).

Project Background

This project recreates the first dungeon of the original NES The Legend of Zelda using Unity. The goal was to faithfully reproduce core mechanics including movement, combat, enemies, item usage, room transitions, and UI systems.

  • Top-down movement + grid room system
  • Enemy AI (Goriya, Aquamentus, etc.)
  • Boomerang mechanics + stun system
  • Heart-based health UI

Technologies Used

  • Game Engine: Unity (URP)
  • Language: C#
  • Architecture: Component-based systems
  • UI: Canvas + Sprite-based health system
